GSLA summary

注：为表述方便，用*表示逆，用'表示转置

Chapter1

这章主要讲了消元和矩阵乘法
高斯-若尔当 消元是一种极好的算法
在求解Ax = b的过程中，完全是线性的，完全符合计算机的操作特点
在讲解过程中，列作为向量引入，完全契合了高中生对空间的理解

Chapter2

这章继续消元的话题，引入了矩阵的逆、转置
重点在于用 高斯-若而当思想 进行消元求逆。
延续消元法，进行LU分解：
A = LU(或A = LDU)
这样，有：A = LUx = b --> Ux = L*b --> Ux = c
方程Ax = b就化为三个过程：

  1. A-->U (消元：A左乘L*; 同时记录下L*)
  2. 解方程：Lc = b（relationship betweenL and L* is wonderful）
  3. 解方程：Ux = c

Chapter3

这章开始进入向量空间、子空间
向量空间的概念：
一切满足一系列规则的向量的集合
子空间的概念：
这里采用集合来描述：
上述集合中同样满足规则的向量构成的集合，且该集合是上一集合的子集。
四大基本子空间：
null space (N), culumn space (C), row space(R), left null space(N’)
容易证明：C(A)⊥N(A') ; R(A)⊥N(A)

Chapter4

正交
（这章是对高中垂直思想的深入，其实不怎么对哈哈哈
1）若两个向量点积为零，则这两个向量正交；
2）若某一向量与某向量空间内的任意向量点积为零，则该向量与该向量空间正交；
3）若某一向量空间内的任意向量与另一向量空间内的点积为零，则这两个向量空间正交；

与高中垂直不同的一点是，在n维度的空间里，当a+b<n时：不存在 a维空间A 与 b维空间B 正交；换句人话说：三维空间里存在两个二维平面垂直，不存在两个二维平面正交，因为2+2 > 3。

投影矩阵公式
先推导2维的情况，再扩展到n维（即2维的情况重复n-1次
结合线性组合的思想，最后得到公式：
举例：向量b要投影到A的列空间：
投影矩阵为：A(A'A)*A'（2D_version: aa'/a'a）

投影矩阵的应用？那必然是最小二乘法（最优解问题
高中的最小二乘法是直线形式

举个例子: 下表是2维平面内的一些点，要拟合最优直线

x 1 2 3
y 1 2 2

即方程组：

即矩阵方程:

1 1 1
2 1 (c,d) = 2
3 1 2

即 Ax = b

但显然：b !∈ C ( A )
这时候，从空间上来看，就是找向量(1,2,3)和(1,1,1)的一个线性组合Ax 使得误差向量e( e = Ax - b )的长度最小
从空间上看，此时自然是当Ax与b在C(A)的投影重合时
此时使用投影矩阵有：Ax = A(A'A)A\*b --> x = (A'A)*b
此时的x就是x的最优解
