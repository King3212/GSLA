GSLA summary

注：为表述方便，用*表示逆，用'表示转置

Chapter1

这章主要讲了消元和矩阵乘法
高斯-若尔当 消元是一种极好的算法
在求解Ax = b的过程中，完全是线性的，完全符合计算机的操作特点
在讲解过程中，列作为向量引入，完全契合了高中生对空间的理解

Chapter2

这章继续消元的话题，引入了矩阵的逆、转置
重点在于用 高斯-若而当思想 进行消元求逆。
延续消元法，进行LU分解：
A = LU(或A = LDU)
这样，有：A = LUx = b --> Ux = L*b --> Ux = c
方程Ax = b就化为三个过程：

  1. A-->U (消元：A左乘L*; 同时记录下L*)
  2. 解方程：Lc = b（relationship betweenL and L* is wonderful）
  3. 解方程：Ux = c

Chapter3

这章开始进入向量空间、子空间
向量空间的概念：
一切满足一系列规则的向量的集合
子空间的概念：
这里采用集合来描述：
上述集合中同样满足规则的向量构成的集合，且该集合是上一集合的子集。
四大基本子空间：
null space (N), culumn space (C), row space(R), left null space(N’)
容易证明：C(A)⊥N(A') ; R(A)⊥N(A)

Chapter4

正交

（这章是对高中垂直思想的深入，其实不怎么对哈哈哈
1）若两个向量点积为零，则这两个向量正交；
2）若某一向量与某向量空间内的任意向量点积为零，则该向量与该向量空间正交；
3）若某一向量空间内的任意向量与另一向量空间内的点积为零，则这两个向量空间正交；

与高中垂直不同的一点是，在n维度的空间里，当a+b<n时：不存在 a维空间A 与 b维空间B 正交；换句人话说：三维空间里存在两个二维平面垂直，不存在两个二维平面正交，因为2+2 > 3。

投影矩阵为：A(A'A)*A'（2D_version: aa'/a'a）

投影矩阵的应用？那必然是最小二乘法（最优解问题
高中的最小二乘法是直线形式

举个例子: 下表是2维平面内的一些点，要拟合最优直线

y 1 2 2

即方程组：

即矩阵方程:

3 1 2

即 Ax = b

但显然：b !∈ C ( A )
这时候，从空间上来看，就是找向量(1,2,3)和(1,1,1)的一个线性组合Ax 使得误差向量e( e = Ax - b )的长度最小
从空间上看，此时自然是当Ax与b在C(A)的投影重合时
此时使用投影矩阵有：Ax = A(A'A)*A'b --> x = (A'A)*A'b
此时的x就是x的最优解

Chapter5

行列式

特性：

  1. I的行列式为1
  2. row change 会变号 —> 1）2） 可推任意置换矩阵的行列式
  3. 每一行都有线性性(加法和数乘，可用box来解释)
  4. 如果两列相同，则行列式为0（交换相同的两行可证
  5. 消元不会对行列式造成影响（换行另说）
  6. 有零行的方阵行列式为零（数乘可证, 亦可由4)得到 )
  7. U or L 方阵的行列式都是对角线元素的乘积 <— 消元可得
    （这里如果A分解成LDU，那直接就是D的行列式）
  8. 行列式为零，方阵奇异
  9. det(BC) = det(B)det(C)
    可推：det(I) = det(A*)det(A) —> det(A\*) = 1/det(A)
    显然，如果A不可逆（det(A) = 0），则 det(A*) 无意义
  10. det(A’) = det(A) （用 7) 易证）

计算方法：

  1. 行列式 or 代数余子式
  2. 消元后求

应用

  1. 求逆矩阵的公式：A* = (1/det(A))C'
    其中C是a的伴随矩阵，C每个位置的元素都是A中对应位置的代数子余式

  2. 克拉默法则：Ax = b —> x1 = det(B1)/det(A)
    其中：A=[a1 a2 ... an]``Bk = [a1...ak-1 b ak+1 ... an]
    推导的关键：xk = [detA1k detA2k … detAnk]b/det(A)
    加粗部分就是Bk的代数余子式

  3. det(A) 可以看作A的各行向量为边围成的box体积（row为边）
    用这个可以解释前面行列式的特性
    我个人是非常喜欢代数几何化的，可能和自己空间想象能力比较强有关

Chapter6

特征量和特征向量

Ax可以看作A对x做的一种变换
那么可能存在这么一些x使得Ax = λx
举个栗子： 若P是个投影到A的矩阵
若x在A的列空间里，Px = x，λ = 1
若x正交于A的列空间，有Px = 0，λ = 0（此为平凡解，一般不考虑 :-) )
既然不考虑平凡解，那也就是说，讨论的A是奇异的

这里的λ就是特征量，符合这样条件的x就是特征向量
会发现：λ 是描述的是变换的程度

特征量和特征向量的求法
从已知的方程开始：Ax = λx (1)
移项：Ax - λIx = 0 (2)
合并：(A-λx)x = 0 (3)
为了解出非平凡解，就要求det(A-λx)=0,这里可以解出n个λ
再将λ代入方程(3)即可解出x
这里有几个点需要注意 :

  1. λ可能出现重解，重解会导致缺base
  2. 代入一个λ，原则上只需解出一个x(作为base)
  3. λ可以是复数
  4. 会发现：x的线性组合就是(A-λx)的零空间

既然有了空间的base（x），把这些base凑一块，就能得到一个方阵X
有：X*AX = Λ``A = XΛX*
推导：先算AX，再把⋀提出来，再左乘X*即可
特性：(A^k)x = X(Λ^k)X*x 显然Λ这里表示的是变换程度

应用
来求个斐波那契数列通项吧
递推：Lk+2 = Lk+1 + Lk
补上：Lk+1 = Lk+1
构造：uk = （Lk+1,Lk)
矩阵方程：

1 1 ak+1 ak+2
1 0 ak ak+1

有：Ax = λx
可解得：
λ1 = (1-√5)/2 ，x1 = （λ1, 1)
λ2 = (1+√5)/2 ，x2 = （λ2, 1)

u1 = ( 1 , 0 ) = x1/√5 + x2/√5
un = A^(n-1)u1 = A^(n-1) (x1/√5 + x2/√5)
= A^(n-1)(x1/√5) + A^(n-1)( x2 /√5)
= λ1^(n-1)x1/√5 + λ2^(n-1)x2/√5
= ( λ1^n / √5 + λ2^n / √5，λ1^(n-1) / √5 + λ2^(n-1) / √5)

∴ Ln = λ1^n / √5 + λ2^n / √5；λ1 = (1-√5)/2，λ2 = (1+√5)/2
当n很大时，可知近似值为λ2^n / √5，λ2 = (1+√5)/2
